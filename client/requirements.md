## Notes
Authentication is handled via Replit Auth (blueprint:javascript_log_in_with_replit).
Use `useAuth` hook from `@/hooks/use-auth` for user state.
API routes are defined in `@shared/routes`.
Images should use Unsplash placeholders where dynamic images aren't available yet.
Color palette should be warm and hopeful (amber/blue or teal/orange).
